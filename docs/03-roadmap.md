# Roadmap — LegalFlow

## 2025

### Q1 — Foundation
- Настройка монорепозитория и инфраструктуры (pnpm, Replit workflows)
- Базовая аутентификация через Clerk
- CRUD для клиентов, дел, задач и календаря
- Загрузка и хранение документов
- Первичная аналитика (дашборд)

### Q2 — MVP Launch
- AI-анализ документов (OpenAI / OpenRouter)
- AI-генерация шаблонов
- Telegram-уведомления для задач и дедлайнов
- Глобальный поиск по CRM
- Русская и английская локализация
- 10 пилотных пользователей

### Q3 — Release 1.0
- Полноценный цикл: клиент → дело → документ → задача → уведомление
- Версионность документов
- Расширенная аналитика (воронка, активность)
- Ролевая модель (owner, attorney, paralegal)
- Подготовка к публичному релизу

### Q4 — Growth
- Интеграция с внешними источниками данных (ЕГРЮЛ, Роспатент) — в режиме пилота
- Email-уведомления (SMTP)
- API для интеграции с CRM
- Улучшенная UX/UI на основе обратной связи

## 2026

### Q1 — Scale
- AI-ассистент с умными подсказками по делам
- Маркетплейс шаблонов (внутренний)
- Enterprise-версия с расширенной RBAC
- Мобильная адаптивная версия (PWA-подход)

### Q2 — Ecosystem
- Интеграция с провайдерами ЭЦП (Госключ / сторонние КЭП)
- Клиентский портал для внешних пользователей
- Двухфакторная аутентификация и аудит действий
- Расширение локализации (дополнительные языки)
